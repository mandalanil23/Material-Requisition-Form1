## 📝 For Site Engineers / Material Coordinators

### How to Submit Material Requisition

#### Step 1: Open Form
Click the link → Form opens ✓

#### Step 2: Fill Basic Info
```
📅 Date          → Auto-filled (today's date)
🔢 MRF Number    → e.g., MRF-2024-001
👤 Requested By  → Your name
📍 Location      → Select your project site
```

#### Step 3: Add Materials
```
1. Click "+ Add Item" button
2. Select Material from dropdown
   → Stock quantity auto-appears
3. Enter Quantity needed
4. Unit auto-fills (Bags, Kg, Piece, etc.)
5. Select Required Date
6. Add remarks (optional)
7. Click "Add Item" to add more
```

#### Step 4: Upload Documents
```
📄 MRF Document  → Attach MRF/requisition file
💰 Quotation 1   → First vendor quotation
💵 Quotation 2   → Second vendor quotation
```

All files MUST be under 5MB

#### Step 5: Submit
```
1. Click "Preview" to check everything
2. Click "Submit MRF"
3. Wait for success message ✓
4. Done! Manager will process it
```

## 📊 Material Status Guide

### Stock Indicators
```
✓ OK         → Plenty in stock (green)
⚠️ LOW       → Low stock (yellow)
❌ OUT       → Not in stock (red)
```

### MRF Status Flow
```
📥 NEW        → Just submitted
⏳ PENDING    → Manager reviewing
✅ APPROVED   → Ready to purchase
❌ REJECTED   → Needs revision
📦 COMPLETED  → Items received
```

## 🆘 Quick Troubleshooting

| Issue | Solution |
|-------|----------|
| **Form won't load** | Clear browser cache, try incognito |
| **Dropdown empty** | Wait 5 seconds, refresh page |
| **Stock not showing** | Select material, wait 2 seconds |
| **Can't upload file** | Check file is under 5MB |
| **Submit error** | Check all fields are filled, retry |
| **Google Sheet not updating** | Contact admin, check internet |

## 🎨 Form Features Explained

### Materials Dropdown
- Shows all available materials
- Auto-updates with latest stock
- Easy to search and select

### Quantity Field
- Enter any quantity
- Auto-calculates against stock
- Shows warning if low stock

### Date Fields
- Simple calendar picker
- Shows when you need items
- Helps with planning

### Preview Button
- See form before submitting
- Print if needed
- Verify all details

### File Uploads
- Multiple file formats supported
- MRF documents (PDF, Doc)
- Quotations (Excel, PDF)
- Photos/Images (JPG, PNG)

## 💡 Tips & Tricks

### Tip 1: Bulk Submissions
```
Need multiple items? 
- Click "Add Item" multiple times
- Fill all at once
- Submit together
```

### Tip 2: Draft Saving
```
Want to save draft?
- Fill form
- NOT YET SUBMITTED
- Use browser history/refresh
- Form data usually persists
```

### Tip 3: Track Your MRFs
```
Keep track of your MRF numbers
- Check submitted MRFs in Google Sheet
- Get confirmation emails
- Reference for follow-up
```

### Tip 4: Estimate Costs
```
Add cost estimates in remarks
- Helps manager plan budget
- Shows cost awareness
- Speeds up approval
```
